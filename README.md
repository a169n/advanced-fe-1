# NLP Short Answer Scoring Board

A frontend-only Next.js (App Router) SPA that simulates a Trello-like pipeline for scoring multilingual student short-answer responses using lightweight heuristics.

## Getting started
1. Install dependencies: `npm install`
2. Run the dev server: `npm run dev`
3. Open `http://localhost:3000` to interact with the board.

State is persisted to `localStorage` under the key `nlp_short_answer_board_state_v1`.
